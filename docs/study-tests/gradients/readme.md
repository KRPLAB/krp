# Estudo do Método do Gradiente Conjugado e Implementação - 12/05/2026

## Contexto

Implementação e validação do algoritmo do Gradiente Conjugado para resolução de sistemas de equações lineares. O método resolve a otimização não-linear "escorregando pela encosta" do paraboloide até encontrar o mínimo.

### Progresso

**Estudo do Algoritmo do Gradiente Conjugado:**
- Revisão matemática de derivadas parciais e vetores gradientes
- Implementação em ambiente isolado (`gradiente.c` e `gradiente_conjugado.c`)
- Validação da teoria através de estrutura padrão do método
